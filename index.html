3. List all global attributes in HTML?

Global attributes are;
  • *id*=> Unique identifier for an element within a document.
  • *class*=> One or more space-seperated values that define the class names of the element.
  • *access key*=> it specifies the shortcut key for an element.
  • *content editable*=> It indicates whether the content of an element can be edited by the user.
  • *style*=> inline style declaration for an element.
  • *data-attributes*=> Custom attributes that can be used to store arbitrary data on an element.
  • *hidden*=> it hides an element from the user.
  • *spellcheck*=> it indicates whether the content of an element should be spell-checked.
  • *tabindex*=> it specifies the order in which an element is focused when navigating with the tab key.
  • *dir*=> it specifies the direction of the text in an element.
  • *lang*=> it specifies the language of the element's content.
  • *title*=> it provides the tooltip for an element.
